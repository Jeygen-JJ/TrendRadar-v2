# JJ 的 AI 资讯系统使用说明

## 当前状态

本地版本已经可以运行：

- 使用 `run-ai-digest.bat` 生成 AI 资讯日报。
- 只抓取 AI 垂直 RSS，不抓综合热榜。
- 暂不启用 AI API、翻译和推送，因此不会产生模型费用。
- 报告模式为 `daily`，展示当天汇总。
- RSS 新鲜度窗口为 7 天，用于扩大早期候选池。

## 每天怎么用

1. 打开项目目录：

   `F:\AI学习\JJ的仓库\JJ的仓库\TrendRadar`

2. 双击：

   `run-ai-digest.bat`

3. 等待报告自动打开。

4. 从报告中挑选 3 到 5 条，建立选题卡片。

## 结果怎么看

- `RSS 源 10/10`：说明当前可用来源全部抓取成功。
- `RSS 命中`：关键词命中的候选资讯数量。
- 理想范围：每天 15 到 30 条候选资讯。
- 如果低于 10 条：说明关键词太严格或当天信息少。
- 如果高于 50 条：说明关键词太宽，需要收紧。

## 当前已停用的来源

- Microsoft Research：Feed 格式当前与解析器不兼容。
- Reddit MachineLearning：当前返回 403。
- Reddit LocalLLaMA：当前容易限流。

后续可以通过 RSSHub 或 Reddit API 重新接入。
